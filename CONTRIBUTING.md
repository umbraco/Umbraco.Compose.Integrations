# Contributing

We welcome contributions to this project! Whether you're interested in fixing bugs, adding new features, or improving documentation, your help is greatly appreciated.

## How to Contribute

1. **Fork the repository**

   Create your own copy of the project by clicking the "Fork" button on GitHub

2. **Create a feature branch**

   Make a new branch for your specific contribution using a descriptive name like `feature/add-new-integration` or `fix/bug-description`

3. **Make your changes**

   Implement your fix, feature, or improvement following the project's coding standards

4. **Write tests**

   Include unit tests or integration tests for your changes when applicable

5. **Follow code standards**

   Ensure your code follows the project's coding conventions and style guidelines

6. **Run tests**

   Verify all existing tests still pass and your changes work as expected

7. **Document your changes**

   Update documentation, comments, or README files as needed to reflect your changes

8. **Submit a pull request**

   Describe your changes clearly and concisely, including the problem being solved and how it's fixed

## Before Adding New Integrations

For new integration components, please create a discussion issue first to ensure:
- The integration fits within the project's scope
- There's alignment on the approach
- No duplicate work is being done
- The team can provide guidance and feedback

## Code Style

This project follows standard coding conventions. Please ensure your code:
- Is well documented
- Follows consistent naming conventions
- Includes appropriate comments
- Is formatted according to the .editorconfig rules

## Commit Guidelines

When contributing to this repository, please follow these commit guidelines:
- Use descriptive commit messages
- Follow conventional commit format
- Keep commits small and focused
- Explain the "why" behind the changes

## Questions or Help

If you have questions or need help getting started, please open an issue in the repository. We're happy to help new contributors get up to speed!
